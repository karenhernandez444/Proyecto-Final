# Proyecto-Final

Este proyecto está enfocado en analizar secuencias genómicas en formato .fna para identificar el palíndromo más largo dentro de los primeros fragmentos de la cadena. La motivación principal surge de la importancia que tienen los palíndromos dentro de la biología molecular, ya que pueden representar sitios de restricción, regiones regulatorias o estructuras que influyen en procesos como la replicación o el plegamiento del ADN. Encontrarlos es una tarea fundamental en bioinformática y existen diversas estrategias algorítmicas que permiten realizar esta búsqueda con distintos niveles de eficiencia.

El código implementa cuatro enfoques diferentes: Fuerza Bruta, Divide y Vencerás, Programación Dinámica y un método Voraz basado en expansión desde el centro. La idea principal es comparar cómo cada uno de estos algoritmos enfrenta el problema de encontrar el palíndromo más largo dentro de una subsecuencia de una cadena de ADN, evaluando tanto la exactitud como el tiempo de ejecución. De esta manera, es posible observar las diferencias reales entre métodos exhaustivos, recursivos, dinámicos y heurísticos aplicados al mismo conjunto de datos.

El programa comienza leyendo un archivo .fna, construyendo una cadena continua que representa la secuencia de ADN. A partir de esta secuencia inicial, se seleccionan subsecuencias de diferentes longitudes para aplicar los algoritmos y permitir una comparación justa entre ellos. Cada método devuelve el palíndromo más largo que encuentra, su longitud y el tiempo que tardó en procesar la misma entrada.

El método de Fuerza Bruta prueba todas las subcadenas posibles y revisa si cada una es palíndroma. Es un enfoque sencillo pero extremadamente costoso cuando la longitud de la cadena crece, por lo que sólo se aplica para tamaños pequeños. El método de Divide y Vencerás divide la cadena en mitades, analiza cada una recursivamente y después revisa la zona central para encontrar posibles palíndromos que crucen ambas partes. La Programación Dinámica utiliza una tabla para identificar subcadenas palíndromas, permitiendo reutilizar resultados anteriores de forma eficiente. Finalmente, el método Voraz se basa en expandir desde cada posible centro de la cadena hacia los extremos, buscando aprovechar la estructura natural de los palíndromos de manera rápida y directa.

Después de aplicar cada algoritmo, el programa mide el tiempo consumido y cuenta cuántas veces aparece el palíndromo encontrado dentro de toda la cadena inicial. Con esta información se construye una comparación completa que se muestra en consola y posteriormente se genera un conjunto de gráficas donde se ilustran los tiempos de ejecución y la longitud de los palíndromos detectados por cada método.

Para ejecutar el proyecto basta con colocar un archivo .fna dentro de la carpeta Datos, asegurarse de tener instalado matplotlib y ejecutar el script principal. El programa cargará la secuencia, realizará las comparaciones y mostrará tanto los resultados como las gráficas. Con esta implementación es posible comprender de manera práctica cómo se comportan distintos enfoques algorítmicos al resolver un mismo problema bioinformático, lo cual permite apreciar las ventajas y desventajas reales de cada técnica cuando se trabaja con datos genómicos reales.
